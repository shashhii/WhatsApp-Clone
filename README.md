## About

Real-time chat application that replicates core WhatsApp features. Built to learn WebSockets, real-time communication, and React state management.

## Tech Stack

React, Node.js, Socket.io, Firebase

## Getting Started

```bash
git clone https://github.com/shashhii/WhatsApp-Clone.git
cd WhatsApp-Clone
```

- Clean and intuitive UI
- Fully functional backend
